# 无障碍生活助手 · 项目说明书

> 参赛项目 · OpenClaw 应用大赛

---

## 项目简介

**无障碍生活助手**是一个面向有需要的人士的 AI 问答平台，帮助用户快速获取残障权益、补贴政策、无障碍出行等实用信息。

项目以 OpenClaw 作为智能运营核心，实现知识库的持续更新、网站内容的自动优化，以及用户问答的智能沉淀——让平台在无人值守的情况下持续生长。

---

## 核心功能

### 1. AI 问答
用户输入问题，系统检索本地知识库后调用大模型生成回答，支持 SSE 流式输出，响应自然流畅。

### 2. 知识库展示与管理
- 前端可浏览、搜索全部知识条目
- 支持按分类筛选（政策 / 就业 / 出行 / 辅具 / 生活技巧 / 健康）
- 管理员可手动新增、删除条目

### 3. 问答自动沉淀入库
每次对话结束后，OpenClaw 自动判断回答是否具有长期参考价值，有则提炼为结构化知识条目写入知识库，实现"越用越聪明"。

---

## OpenClaw 的角色

本项目将 OpenClaw 作为**后台智能运营引擎**，承担三类核心工作：

### 🔄 持续更新知识库

OpenClaw 定期执行 `bank-crawler` skill，自动完成：

1. **爬取政策网站**：抓取中国残联官网、国务院、各省市政府网站的最新政策文章
2. **AI 提炼入库**：调用大模型将原始文章提炼为结构化知识条目（标题、摘要、可操作建议、适用人群）
3. **去重写入**：按标题 + URL 双重去重，避免重复条目
4. **小红书数据导入**：通过 `import_xhs.py` 将爬取的真实用户经验导入知识库

当前知识库已收录 **53 条**权威条目，覆盖政策、就业、出行、辅具、健康等方向，并持续增长。

```
知识库分类分布（2026-03-18）：
  政策: 25条   就业: 11条   生活技巧: 6条
  出行:  4条   辅具:  3条   健康:     2条
```

### 🛠 优化网站内容

OpenClaw 通过对话与代码能力，协助完成：

- **前端迭代**：根据用户反馈修改 UI 样式、交互逻辑、无障碍适配（大字体、高对比度、键盘导航）
- **后端优化**：改造 API 接口、修复 SSE 流式解析、新增问答沉淀功能
- **数据脚本维护**：编写并维护 `import_xhs.py`、`crawl_policy.py` 等数据处理脚本
- **日志分析**：读取 `server.log` 排查异常，优化服务稳定性

### 💬 运营网站

OpenClaw 作为平台的"运营助手"，负责：

- **撰写宣传文案**：根据知识库内容生成适合各平台（小红书、公众号、飞书）的推广文案
- **问答质量监控**：定期检查 `qa_history.json`，识别高频问题，补充对应知识条目
- **数据统计播报**：读取访问统计，汇报 PV、活跃时段等运营数据
- **内容审核建议**：对新入库条目进行语气审查，确保表达温和、无歧视性用语

## 技术架构

```
用户浏览器
    │
    ▼
Flask 后端 (server.py)
    ├── /api/chat        SSE 流式问答
    ├── /api/kb          知识库 CRUD
    ├── /api/kb/search   全文搜索
    └── /api/stats       访问统计
    │
    ├── knowledge_base.json   本地知识库（53条）
    ├── qa_history.json       问答历史
    └── stats.json            访问统计

OpenClaw（后台运营）
    ├── bank-crawler skill    爬取 + AI提炼 + 入库
    ├── import_xhs.py         小红书数据导入
    └── 对话驱动              前端优化 / 文案撰写 / 数据分析
```

**技术栈**：Python · Flask · SSE · 本地 JSON · OpenAI 兼容 API（kimi-k2-thinking）

## 项目亮点

**1. 知识库自生长**
OpenClaw 定期爬取 + AI 提炼，知识库无需人工维护即可持续扩充，真正实现"越用越聪明"。

**2. 问答即贡献**
用户每次提问，系统自动判断是否有新知识价值并沉淀入库。用户在获得帮助的同时，也在为后来者积累经验。

**3. 全链路 AI 驱动**
从数据采集、内容提炼、问答服务，到网站优化、运营文案，全部由 AI 参与完成，人工介入极少。

**4. 专注垂直领域**
系统提示词、知识分类、适用人群标签均针对无障碍/残障服务领域深度定制，回答质量远优于通用 AI。
